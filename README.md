# Theme Switcher — Zustand Global State

Build a Light/Dark/Ocean theme toggle using Zustand for global state. One store controls the entire app's look, with localStorage persistence.

## Quick Start

```bash
git clone https://github.com/LoisBN/fpp-theme-switcher.git
cd fpp-theme-switcher
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) to see the app.

> You will also need to install Zustand: `npm install zustand`

## Tech Stack

- React Router v7 (framework mode)
- Zustand (global state)

- TypeScript

## Project Structure

```

```
